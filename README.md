# Todo List Backend

## Description

Spring Boot backend for Todo List application.

## Installation

```bash
mvn clean install
```

## Usage

```bash
mvn spring-boot:run
```

## Docker execution

It provides a PostgreSQL database and the service. To run it, execute: 

```bash
docker-compose up
```

## Swagger

Swagger is available at: http://localhost:8080/swagger-ui/index.html
